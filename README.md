# SmartFileOrganizer

SmartFileOrganizer es una aplicación de escritorio que organiza automáticamente archivos de una carpeta, clasificándolos según su tipo.

## Características

- Organización automática de archivos
- Clasificación por extensión
- Configuración personalizable
- Historial de acciones realizadas
- Interfaz gráfica simple

## Instalación

Descarga el .Zip en Releases y extraelo


### Usando el ejecutable

1. Descargar el archivo:

`SmartFileOrganizer.exe`

2. Abrir el programa.

No es necesario tener Python instalado.

### Ejecutar desde el código fuente

Requisitos:

- Python 3.x

Instalar las dependencias:

```bash
pip install -r requirements.txt
